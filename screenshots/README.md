# Scratch-ticket data screenshots

Reference PNG screenshots of PA scratch-off game data (e.g. lottoedge prize-count
tables). These are a point-in-time manual capture used for cross-checking the
tracker against an outside source.

> Note: the live tracker does **not** depend on these — it pulls the same per-tier
> data automatically from PA's official Bulletin + Remaining pages. These are kept
> as a reference / cross-check set.

Drop game screenshots here named by game number when possible (e.g.
`1774-500-blowout.png`) so they're easy to match to a game.
